# Historical comparison library

`historical-episodes.json` is a compact qualitative reference library used by the daily learning raport.

The AI may select only episodes from this file. It must explain:

- what looks similar;
- what is materially different;
- which current signals would strengthen the comparison;
- why the comparison might fail.

The library deliberately does not contain invented or loosely remembered return statistics. Historical comparisons are hypotheses for further research, not forecasts.

For quantitative historical work, verify claims against primary datasets such as FRED/ALFRED, BLS, BEA, central-bank releases, and company filings.
