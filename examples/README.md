# Example output

These are real reports generated by the tool, included so you can see what it
produces without running the full pipeline yourself.

| File | What it is |
| --- | --- |
| `example-daily-brief-2026-06-30.md` | A daily market brief in Markdown (the readable text behind the PDF) |
| `example-daily-raport-2026-06-30.pdf` | The rendered daily PDF report |
| `example-weekly-raport-2026-07-01.pdf` | The rendered weekly learning review PDF |

All content is educational. Historical parallels are treated as hypotheses,
not forecasts, and the tool does not invent historical returns.
